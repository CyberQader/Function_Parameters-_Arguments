# 🖋️ Smart Name Formatter

سكربت بايثون بسيط وذكي لاستقبال الأسماء الثلاثية وتنسيقها بشكل احترافي باستخدام الـ **F-Strings**.

## 🛠️ ماذا يفعل الكود؟ (Logic)
* **`strip()`**: إزالة أي مسافات زائدة من البداية والنهاية.
* **`capitalize()`**: تحويل أول حرف من كل اسم إلى كبير (Capital).
* **`:.1s`**: حركة احترافية لاستخراج **أول حرف فقط** من الاسم الأوسط (Middle Initial).
